; Yes, you can absolutely get worldwide, real-time soccer events via APIs.

Here are some strong options (all with global coverage):

Sportmonks Football API – “Worldwide plan” and scores API with livescores and events for over 2,000 leagues worldwide, including top and lower divisions across all continents.

API-Football (apifootball.com) – Affordable football API with livescores for cups, leagues, and international matches covering major federations all over the world; you can inspect the full country/league coverage list.

Live Score API (Football API) – Provides live scores, fixtures, standings, history data and match events, with competitions in 200 countries and 10 international federations (about 460 competitions total).

Soccerdata API – Live scores, match events, stats, lineups, and odds for 125+ leagues in 95 countries, with real-time updates and sample code for fetching livescores.
​

Sportradar Soccer API – Enterprise-grade soccer feed with real-time data and rich stats across more than 900 competitions worldwide, including play-by-play style timelines and live standings.
